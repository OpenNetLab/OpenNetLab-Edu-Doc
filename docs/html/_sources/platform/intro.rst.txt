===============================
OpenNetLab for Education Intro
===============================

在进行实验之前，需要先导入onl-edu包，具体操作如下：

.. code-block:: text

    $ pip3 install onl-edu

如果导入失败，可能是由于网络的原因导致，需要更换为国内镜像源，具体操作如下：

.. code-block:: text

    $ pip3 install onl-edu -i https://pypi.tuna.tsinghua.edu.cn/simple
